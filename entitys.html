Projeto Entity’s Bazzar


Dado as orientações do professor, todos os alunos escolheram temas e sortearam entre eles com a ideia de fazer um site institucional com base no tema sorteado.

O tema que eu tirei foi de um jogo de terror bastante famoso, onde fiz o seguinte site com HTML, CSS e um sistema de login, includes, requires e manipulação de dados em PHP.
